# learn-sql-python

Small learning project to practice raw SQL (BigQuery) and basic Python analysis.

## Goals
- Build comfort with SQL: `SELECT`, `WHERE`, `GROUP BY`, `JOIN`, `HAVING`, `ORDER BY`
- Use Python to fetch query results and do light analysis
- Answer real questions using public data (Hacker News)

## Structure
- `src/` Python code
- `queries/` store `.sql` files you write
- `PLAN.md` checklist and milestones
- `references/` notes and research frameworks

## How it will work
1) Write SQL in `.sql` files (explore, filter, aggregate)
2) Run queries in BigQuery
3) Pull small result sets into Python for labeling/summaries
4) Interpret results in a short write-up

## Next step
Start with Step 1 in `PLAN.md` and log progress in the Status section.
